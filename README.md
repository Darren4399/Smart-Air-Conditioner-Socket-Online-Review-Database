# Smart-Air-Conditioner-Socket-Online-Review-Database

This repository contains a cleaned dataset with 3,254 user reviews of smart air conditioner socket products collected from platforms such as Taobao and JD.com.


This dataset is designed to support the following tasks:
-Customer Requirement Extraction
-Function Transformation and Mapping
-Sequence labeling
-Named Entity Recognition
